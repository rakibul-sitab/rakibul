Case study 2 : Analysing customer retention percentage using excel

This is a transnational data set which contains all the transactions occurring between 01/12/2010 and 09/12/2011 for a UK-based and registered non-store online retail.The company mainly sells unique all-occasion gifts. Many customers of the company are wholesalers.We want to understand the cohort s retention analysis using excel.

CASE STUDY

Case study 2 : Analysing customer retention percentage using excel

May 16, 2021

This project is to advise marketing experts about customer retention using excel.

Key procedures

Excel column:

Purchase rate:

Purchase rate calculates the repetition of purchase of a unique customer.The formula used to calculate purchase rate: =IF(A2=A1;I1+1;1)

Joint date:

Joint date calculates the first purchase of a unique customer.The formula used to calculate joint date: =IF(I2=1;B2;J1)

Cohorts:

Joint date month and year have been extracted in the cohorts column.The formula used to calculate cohorts: =TEXT(J2;“mmm-yyy”)

Age by days:

The days have been calculated by subtracting first date purchase to purchase date of each customer.The formula used to calculate joint date: =DATEDIF(J2;B2;“d”)

Age by months: The age by months column have been calculated by referencing below column:

After that pivot tables have been created. Joint data column have been placed in the rows and age by months column in the column fields. In the values field, a customer id column is placed and a count function has been used. From the value field of the pivot table,Percentage of retention have been calculated using below navigation.

Field settings > show data as > %of >base field(Age by months) > Base item(0)

To make a retention heat map,

select Conditional formatting > color scales

Analysis:

The retention cohort analysis represents Customer retained across Cohorts. We can read it as following:

  1. Index value represents the Cohort
  2. Columns represent the number of months since current Cohort
  3. For example the value at index Dec 10 in column no. 15 is 24.4% of customers from cohort Dec 10 were retained in 12th Month.In other word, only 6742 users retained from 175123 users after 12 months.

Suggestions to improve more revenue:

Dec 2010 was the highest purchasing month. AS we saw that there is a sharp drop-off in the second month (indexed as 1) already, on average around 62% of customers didn’t come back in the second month.We could investigate the landing/purchase/checkout page in the website.Also have to change marketing strategy.
