PRACTICUM100 PROJECTS

Project 1 : Report preparation for a bank's-loan division.

February 2, 2021

This project is to prepare a report for a bank’s loan division.I was provided a dataset from Practicum100 which is based on some data on customers’ credit worthiness. I found out if a customer’s marital status and number of children has an impact on whether they will default on a loan.My report have been considered when building a credit score for a potential customer. A credit score is used to evaluate the ability of a potential borrower to repay their loan.I complete the entire data analysis process by sharing the findings.

Description of the data column which have provided by the bank.

Columns Description
children : the number of children in the family
days_employed : how long the customer has been working
dob_years : the customer’s age
education : the customer’s education level
education_id : the identifier for the customer’s education
family_status : the customer’s marital status
family_status_id : the identifier for the customer’s marital status
gender : the customer’s gender
income_type : the customer’s income type
debt : whether the customer has ever defaulted on a loan
total_income : monthly income
purpose : the reason for taking out a loan

Step 1 - Understanding the data :

In the beginning of this project, the data file have been looked at deeply and find out the general information. The data set has been loaded into the credit_score data frame and 12 columns have been found.

Step 2 - Data preprocessing :

The days_employed column and total_income column have null values which have been found by using isnull().sum() method.The days_column value has also negative values which have been converted to positive values using abs() because the working days could not be negative.

days_employed and total_income column are quantitative variables. Missing values for such variables are filled in with representative values: values that characterise the dataset being analysed. The mean or the median can be taken as the characteristic value for a dataset. So, the outliers have been checked. The data has significant outliers so the median() method has been selected and null values have been filled with median according to income_type and education column.

children column also have outlier(=20) value.Using describe() method, 25% quantile value(=0) and 75% quantile value(=1) have calculated. Negative children number have been converted to positive number and outliers have removed using 1.5*IQR method.

After that education_id and family_status_id column has been converted from int data type to object data type using the try-except method but didn’t consider for this project work.

The loan purpose column have been categorised using lemmatisation.The category are

  1. real estate
  2. car
  3. education
  4. house
  5. property
  6. wedding

The Total income column have been categorised using the following rules:

  • ‘low income non defaulters’ for total_income <= 17000 and default status 0
  • ‘low income defaulters ' for total_income <= 17000 and default status 0
  • ‘low middle income non defaulters ' for 17000< income <= 23000
  • ‘low middle income defaulters ' for 17000< income <= 23000
  • ‘high middle income non defaulters’ for 23000< income <= 32000
  • ‘high middle income defaulters’
  • ‘high income non defaulters for all other cases
  • ‘high income defaulters’

Step 3-The main finding of this analysis are:

  1. Higher number of children have higher loan default ratio percentage and for lower vice versa.
  2. Unmarried people and civil partner have higher loan default percentage ratio and widow people have lower loan default percentage ratio.Unmarried people have more default loan percentage than married people.
  3. All income level non defaulters have chance to repay loan on time.
  4. Car and education loan have bad review to repay loan in time.House loan is at safest position.

Suggestion for preparing credit score : After analysing the whole data sets and calculating defaulters-loan percentage ratio , I strongly recommend to give house loan to potential customers who are widow working people having no children and no debts.
