Project 9 : Investigate user behaviour for the company's app.

We work at a startup that sells food products. we need to investigate user behaviour for the company's app. The users are split into three groups: two control groups get the old fonts and one test group gets the new ones. We've to find out which set of fonts produces better results.

Project description: The designers would like to change the fonts for the entire app, but our managers are afraid the users might find the new design intimidating. To make a decision we did an A/A/B test and funnel analysis.
